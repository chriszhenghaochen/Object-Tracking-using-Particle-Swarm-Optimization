# Object-Tracking-using-Particle-Swarm-Optimization

## Overview
This is a ***A new Weight Adjusted Particle Swarm Optimization System*** for Real-time Multiple Object Tracking

## Library Requirement
1. Java > 7
2. OpenCV > 2.4.9
